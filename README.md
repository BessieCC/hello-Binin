# hello-Binin My dating day
## 2025-01-17 玉鸟集+Blue Frog
下午逛了文艺街区———**玉鸟集** 
和天目里有点相似，但更生活化，没有特别多的网红街拍打卡。
那天有一条流动摊位展出，出售来自各个地方的特色美食，各式各样的特产琳琅满目。街边有很多很有意思的小店。
其中有一家书店很有意思，它的整体设计就像一个粮仓，建筑与岛型的地块完美结合，设计师给它取名为谷仓，将储存粮食的用意运用于此，想向我们表达的是，精神食粮也是我们所需要的一类粮食。真的有被震撼🫨到！  ![谷仓书店](https://raw.githubusercontent.com/BessieCC/hello-Binin/main/IMG_2273.jpeg)
晚上我们去吃了**蓝蛙**，久闻大名从未尝试，招牌汉堡汁水很足，肉类也鲜嫩多汁。我最爱的蘑菇汤！！！！依旧如此美味～🤤 ![dinner](https://raw.githubusercontent.com/BessieCC/hello-Binin/main/IMG_2311.jpeg)
那一刻，被爱意裹满着的我，不用想太多，成为了拥有此刻便无比幸福的大人❤️![love](https://raw.githubusercontent.com/BessieCC/hello-Binin/main/IMG_2275.jpeg)
